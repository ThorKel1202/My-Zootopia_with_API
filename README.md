# Das MSIT-Projekt "My Zootopia" von Thorsten Kelterborn

Hier kannst du dir Informationen zu allen möglichen Tieren holen. Was sie fressen, 
wie ihr wissenschaftlicher Name lautet, welche Farbe sie haben, wo sie vorkommen
und welcher Typ sie sind.

## Installation

Um dieses Projekt zu installieren, klone einfach das Repository und installiere die Abhängigkeiten aus der requirements.txt mit `pip`

## Nutzung

Um dieses Projekt zu nutzen, führe folgenden Befehl aus – `python animals_web_generator.py`
und folge dem Link (http://127.0.0.1:5000) im Terminal.
